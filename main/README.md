# Milestone-1-CS361


User story 1:
As a user, I want to have my personal data stored automatically in a database so that when I close the web app, it does not disappear forever

User story 2:
As a user, I want to have my personal data sent from the database to the web app, so that I can see it displayed inside the app.

User story 3:
As a user, I want to see changes or additions to my data to be reflected within the app, so that I can see any differences in real time.
